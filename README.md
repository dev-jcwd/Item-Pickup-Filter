# Item Pickup Filter

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-success.svg)

**Item Pickup Filter** is a Quality-of-Life (QoL) mod that gives you complete control over your inventory. Stop filling your bags with trash, dirt, or low-tier loot! This mod allows you to configure exactly which items you want to pick up automatically and which ones you want to ignore.

## ✨ Features

* **Whitelist & Blacklist Modes:** Choose whether to *only* pick up specific items (whitelist) or pick up everything *except* specific items (blacklist).
* **In-Game Configuration UI:** Easily add or remove items from your filter list using a clean, intuitive in-game menu.
* **Real-time Filtering:** Changes to your filter apply instantly. No need to reload the game or your save file.
* **Mod Support:** Fully compatible with custom items added by other mods.
* **Rarity Filtering (Optional):** Filter items based on their rarity/tier, not just their specific names.

## 📥 Installation

1. Ensure you have the required Mod Loader installed (e.g., Forge, Fabric, or BepInEx, depending on the base game).
2. Download the latest `ItemPickupFilter.jar` (or `.dll`) from the [Releases](../../releases) page.
3. Drop the downloaded file into your game's `mods` or `plugins` folder.
4. Launch the game!

## ⚙️ Usage

1. Once in-game, press the default hotkey **[`P`]** to open the Item Pickup Filter menu. *(This hotkey can be changed in the game's control settings).*
2. Select your desired mode: **Whitelist** or **Blacklist**.
3. Search for items and click them to add them to your active filter list.
4. Close the menu, and your new pickup rules will be applied automatically!
